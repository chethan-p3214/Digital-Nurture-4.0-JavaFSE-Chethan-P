Spring Data JPA Quick Example - orm-learn Project

Steps:
1. Import project into Eclipse using Maven > Existing Maven Projects.
2. Create schema 'ormlearn' in MySQL using `init.sql`.
3. Update DB credentials in application.properties if needed.
4. Run OrmLearnApplication.java.
5. Check console logs to verify Spring Boot and Hibernate setup.

Dependencies used:
- Spring Boot DevTools
- Spring Data JPA
- MySQL Driver
